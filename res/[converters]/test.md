# Markdown Converter

Немного подзадолбало пользоваться лонгформом для склеивания и очистки маркдаун файлов, а потом вручную конвертировать в fb2 и другие форматы. Поэтому решено написать большой скрипт, который будет автоматически собирать разрозненные файлы в нужный формат.
## Сложности реализации

- Произведение может быть размещено в одном или нескольких файлах.
	- Если произведение размещено в одном файле:
		- Заголовок произведения удобно записывать в начале этого файла
	- Если произведение размещено в нескольких файлах:
		- Заголовок удобно записывать в отдельном файле.
- Произведение может состоять из нескольких частей:
	- Часть может содержать секцию без заголовка — что-то вроде введения. Например, эпиграф к части. В таком случае эту секцию удобнее размещать в файле с заголовком
