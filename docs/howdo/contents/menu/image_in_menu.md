# 4.7. Как вставить картинки в меню?

**В:** Как вставить картинки в меню?
**О:**
Просто укажите путь к файлу картинки следующим значением в кортеже.

```qsp
%меню_отвёртка[0] = ['Осмотреть', 'info_screw', 'img/screw.png']
```

При этом название меню можно не писать, но оставить пустые кавычки, и пункт меню будет выводиться без надписи:

```qsp
%меню_отвёртка[0] = ['', 'info_screw', 'img/screw.png']
```
