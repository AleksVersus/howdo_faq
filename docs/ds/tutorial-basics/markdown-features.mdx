# Возможности Markdown

Docusaurus поддерживает **[Markdown](https://daringfireball.net/projects/markdown/syntax)** и несколько **дополнительных функций**.

## Вводная часть

Документы Markdown имеют метаданные в верхней части, называемые [Вводная часть](https://jekyllrb.com/docs/front-matter/):

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: My document title
description: My document description
slug: /my-custom-url
---
// highlight-end

## Markdown heading

Markdown text with [links](./hello.md)
```

## Ссылки

Поддерживаются обычные ссылки Markdown, использующие пути URL или относительные пути к файлам.

```md
Let's see how to [Create a page](/create-a-page).
```

```md
Let's see how to [Create a page](./create-a-page.md).
```

**Результат:** Давайте посмотрим, как [Создать страницу](./create-a-page.md).

## Изображения

Поддерживаются обычные изображения Markdown.

Вы можете использовать абсолютные пути для ссылок на изображения в статическом каталоге (`static/img/docusaurus.png`):

```md
![Docusaurus logo](/img/docusaurus.png)
```



Вы также можете ссылаться на изображения относительно текущего файла. Это особенно полезно для размещения изображений рядом с файлами Markdown, используя их:

```md
![Docusaurus logo](./img/docusaurus.png)
```

## Блоки кода

Блоки кода Markdown поддерживаются с подсветкой синтаксиса.

````md
```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```
````

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```

## Предупреждения

Docusaurus имеет специальный синтаксис для создания предостережений и выносок:

```md
:::tip Мой совет

Используйте эту замечательную опцию функции

:::

:::danger Будьте осторожны

Это действие опасно

:::
```

:::tip  Мой совет

Используйте эту замечательную опцию функции

:::

:::danger Будьте осторожны

Это действие опасно

:::

## Компоненты MDX и React

[MDX](https://mdxjs.com/) может сделать вашу документацию более **интерактивной** и позволяет использовать любые **компоненты React внутри Markdown**:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`You clicked the color ${color} with label ${children}`)
    }}>
    {children}
  </span>
);

This is <Highlight color="#25c2a0">Docusaurus green</Highlight> !

This is <Highlight color="#1877F2">Facebook blue</Highlight> !
```
