Базовые системные переменные

 Top  Previous  Next
NOSAVE - если её значение отлично от 0, то сохранение состояния игры пользователем невозможно.

 

DISABLESCROLL - если значение переменной не равно 0, то запрещает автопрокрутку текста при его выводе в основное или дополнительное окно описания локации.

 

DISABLESUBEX - если значение переменной не равно 0, то запрещает использование "подвыражений" в тексте (например, значением '<<5+6>>' будет строка '<<5+6>>', а не '11').

 

DEBUG - если значение переменной не равно 0, то отключается проверка идентификатора игры при загрузке состояния. Полезно для отладки.
