Quest Navigator

Плеер Quest Navigator создаётся как следующий шаг развития платформы QSP, на смену устаревшим классическому плееру и AeroQSP.

Принципиальные отличия

  1. Содержимое отрисовывается средствами браузера, благодаря этому есть полная поддержка HTML, CSS и Javascript
  2. Оформление отделено от кода игры
  3. Облегчено портирование плеера благодаря перемещению платформонезависимой части логики в общий Javascript-фреймворк
  4. Поддержка мобильных платформ (Android, iOS)

Доступен на платформах

  • Android
  • iOS

Будущие платформы

  • Windows Phone 8 (разработка не начата)
  • Десктопные версии Windows (в разработке)1)
